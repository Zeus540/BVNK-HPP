## Description
A demo hosted payment portal built in react that allows users to select a crypto currency in which they would like to pay a merchant

## Supported Currencies
1. Bitcoin
2. Etheruim
3. Litecoin

## Setup
    git clone
    cd BVNK-HPP
    npm install
    npm run dev
